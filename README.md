### Prerequisites
- Flutter SDK
- Dart SDK
- Android Studio / VS Code

### Installation
1. Clone the repository
```bash
git clone https://github.com/shakeelkhalid1913/smart_quiz.git
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```
